Tiga Cara Menambahkan CSS

1. CSS Inline

Ini adalah contoh CSS inline menggunakan atribut style. CSS inline memiliki prioritas tertinggi.

2. CSS Internal

Ini adalah contoh CSS internal menggunakan tag style di bagian head. CSS internal baik untuk halaman yang memiliki style unik.

3. CSS External

Ini adalah contoh CSS external yang diimpor dari file styles.css. CSS external adalah cara terbaik untuk maintenance website besar.

Prioritas CSS

Teks ini memiliki beberapa style yang bertabrakan. CSS inline akan menang karena memiliki prioritas tertinggi.
