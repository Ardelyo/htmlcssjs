## 📖 Tentang Repository Ini

Repository ini adalah jalur pembelajaran terstruktur untuk calon developer web yang ingin menguasai fundamental pengembangan web. Kurikulum dirancang dengan hati-hati untuk memberikan pengalaman belajar yang progresif, dimulai dari konsep HTML dasar hingga pemrograman JavaScript yang lebih kompleks.

### 🎯 Tujuan Pembelajaran

Setelah menyelesaikan kursus ini, kamu akan mampu:
- ✅ Menguasai struktur HTML5 semantik dan praktik terbaik
- ✅ Membuat desain responsif dan menarik dengan CSS3
- ✅ Mengimplementasikan fungsionalitas interaktif menggunakan JavaScript
- ✅ Membangun proyek web lengkap dari nol
- ✅ Memahami alur kerja pengembangan web modern

## 📖 Cara Menggunakan Repository Ini

### 🔄 **Metodologi Pembelajaran**

1. **📋 Pembelajaran Berurutan**
   - Ikuti modul sesuai urutan nomor (01 → 02 → 03 → 04)
   - Selesaikan setiap pelajaran sebelum melanjutkan
   - Bangun pengetahuan fundamental yang kuat

2. **📚 Pelajari Setiap Materi**
   - Baca `PENJELASAN.md` untuk konsep teoretis
   - Periksa contoh kode dengan komentar detail
   - Pahami "mengapa" di balik setiap implementasi

3. **🛠️ Praktik Langsung**
   - Modifikasi contoh kode yang ada
   - Kerjakan "Tugas Kecil" di setiap materi
   - Bereksperimen dan jangan takut salah (cara terbaik belajar!)

4. **🔍 Review dan Perkuat**
   - Kunjungi kembali konsep sebelumnya secara berkala
   - Hubungkan pembelajaran baru dengan modul sebelumnya
   - Ajukan pertanyaan dan cari klarifikasi

### 📁 **Konvensi Struktur File**

Setiap folder pelajaran berisi:
- `PENJELASAN.md` - Penjelasan komprehensif dan teori
- `index.html` - Contoh kode langsung dengan komentar
- `style.css` - Contoh styling (jika berlaku)
- `script.js` - Fungsionalitas JavaScript (jika berlaku)
- `assets/` - Gambar, font, dan sumber daya lainnya
