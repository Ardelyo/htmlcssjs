Langkah-langkah Membuat Kopi

  1. Siapkan air panas
  2. Masukkan kopi ke dalam gelas
  3. Tuangkan air panas
  4. Tambahkan gula sesuai selera
  5. Aduk hingga merata

Istilah HTML Dasar

HTML
HyperText Markup Language - bahasa markup standar untuk membuat halaman web.
Tag
Elemen HTML yang diapit oleh tanda kurung sudut, seperti <p> untuk paragraf.
Atribut
Informasi tambahan yang diberikan pada tag HTML, seperti class atau id.
